Overview

Learn about SQL Server on Linux and watch demos on how to get started today.

High Availability and Security

Learn about what we're planning for HADR and what security capabilities are available for SQL Server on Linux.

Tools

We'll show you how to use SQL Server Management Studio (SSMS) and SQL Server Data Tools (SSDT) with SQL Server on Linux as well as new cross-platform SQL tooling experiences that are available now in Visual Studio Code on Linux, macOS, and Windows.

Connectivity

Learn how to get started with programming apps in PHP and Java connecting to SQL Server on Linux. PHP and Java both have open sourced connectors that can be used by applications to connect to SQL Server anywhere (on-premises or in the cloud).

How We Did it

Hear about the technical details and the thought process behind SQL on Linux and then watch a demo showing how the team is debugging SQL Server on Linux with existing and newly built tools.

How We Did it Part 2

Watch the sequal to The How where you'll see exactly what happens, and actually show the code, as we transition between the Linux world and the Windows world through the host SQLPAL technology

Demos

Watch how to install, configure and get an application running on Red Hat Enterprise Linux.

Visual Studio Code Extension

With this extension, you can connect to SQL Server, create and manage connection profiles, write T-SQL scripts with IntelliSense, and much more.
